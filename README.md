# TODO APP - CLIENT

### Tech Stack

- React & Hooks
- Styled-Components for styling
- Axios for fetching data

### How to start

- Git clone

```bash
git clone [ url ]
```

- Change directory and install dependency packages

```bash
cd todo-app-client && npm install
```

- Run development mode

```bash
npm start
```

- Build

```bash
npm run build
```
